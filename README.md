# COVID-19 Global Data Analytics Project
Author: Rocky Mavie ADZABA EMVO

## Overview
This repository contains an end-to-end COVID-19 data analytics project focused on deaths and vaccination data. It is designed for use with Snowflake SQL for data warehousing and Power BI for visualization.

## Contents
- `data/` - CSV files (upload your CovidDeaths and CovidVaccination CSVs here)
- `snowflake_sql/` - DDL and analysis scripts for Snowflake
- `powerbi/` - Instructions and suggested report structure for Power BI
- `images/` - Example charts generated from the provided data (PNG previews)
- `notebooks/` - Simple Python script for EDA

## How to use
1. Upload the CSV files to `data/` or to your Snowflake user stage.
2. Run the Snowflake scripts in `snowflake_sql/` to create tables and load data.
3. Build the Power BI report following `powerbi/POWERBI_INSTRUCTIONS.txt`.
4. Use the `images/` folder as preview images for your GitHub README.
